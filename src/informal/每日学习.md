1. 算法：移除链表元素

```js
var removeElements = function (head, val) {
  let h = new ListNode(-1, head)
  let res = h;
  while(h.next) {
    if (h.next.val === val) {
      h.next = h.next.next
      continue
    }
    h = h.next
  }
  return res.next;
};
```
注意点：
+ 需要创建一个虚拟头结点，方便删除头部元素
+ while循环，判断h.next是否存在
+ `h.next = h.next.next` 操作完成后，需要contine, 不能再继续往后移动指针了，因为可能删除后的链表，下一个元素还是目标元素
