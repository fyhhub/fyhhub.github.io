# GET 和 POST 区别

| 特性         | GET                                            | POST                                                           | 备注                                 |
| ------------ | ---------------------------------------------- | -------------------------------------------------------------- | ------------------------------------ |
| 是否可以缓存 | :heavy_check_mark:                             | :x:                                                            |                                      |
| TCP 包       | 一个，header 和 data 会一起发送                | 两个，header 和 data 先后发送，会先返回 100 状态码，后返回 200 | 注意：Firefox 的 post 请求只发送一次 |
| 长度限制     | 数值取决于浏览器和服务器的限制，最长 2048 字节 | post 无限制                                                    |                                      |
| 数据位置     | 数据拼接在 url 参数上                          | 请求体                                                         |                                      |
| 浏览器回退   | 不会重新请求                                   | 重新请求                                                       |                                      |
| 安全         | 数据携带在参数上不安全                         | 安全                                                           |                                      |
