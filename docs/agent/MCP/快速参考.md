## 常见操作

### 创建自定义 Agent（opencode.json）

```jsonc
{
  "agent": {
    "my-agent": {
      "description": "我的自定义 Agent",
      "mode": "primary",
      "model": "anthropic/claude-3-5-sonnet",
      "temperature": 0.7,
      "steps": 30,
      "permission": {
        "*": "allow",
        "bash": "ask",
        "external_directory": "deny"
      }
    }
  }
}
```

### 创建子 Agent

```jsonc
{
  "agent": {
    "reviewer": {
      "description": "代码审查专家",
      "mode": "subagent",
      "permission": {
        "read": "allow",
        "grep": "allow",
        "*": "deny"
      }
    }
  }
}
```

### 调用子 Agent（TaskTool）

```
@reviewer 请帮我审查 src/auth/login.ts 文件
```

或通过命令：

```typescript
// 内部会创建 SubtaskPart，触发 TaskTool
parts: [{
  type: "subtask",
  agent: "reviewer",
  prompt: "请帮我审查...",
  description: "代码审查"
}]
```

      continue
    }

## 错误处理

| 错误类型 | 说明 | 处理 |
|---------|------|------|
| `DeniedError` | 权限被拒绝 | 规则配置问题 |
| `RejectedError` | 用户拒绝 | 正常流程 |
| `CorrectedError` | 用户拒绝但有建议 | 采纳建议 |
| `Session.BusyError` | 会话忙碌中 | 等待或取消 |
